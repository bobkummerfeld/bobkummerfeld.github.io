1.3 Variables¬∂

Variable names¬∂

Python variable names have to follow some rules. The name must start with upper/lower case alphabetic or an underscore. It can only continue with alphabetics/numbers/underscore.

Type hints¬∂

Python is a dynamically typed programming language. Variable names can label any type of data. However, you can give a hint about what type of data is being labelled.
