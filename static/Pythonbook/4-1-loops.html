4.1 Loops¬∂

Python for loop¬∂

We very often want to process elements of lists and other data structures. The Python for loop allows you to step through a list and access each element.


 

 

 

 

 

 

Python BREAK statement¬∂

Sometimes you want to interrupt a loop like for or while, and jump out of it to the end of the loop. You can do this using the ‚Äúbreak‚Äù statement.


 

 

 

 

 

 

Python CONTINUE statement¬∂

Some loops need to be continued. In this case you use the Python continue statement that will jump to the beginning of the loop again.


 

 

 

 

 

 

Python While loop¬∂

In a program you very often want to do a repeated action until some condition is true. This might be prompting a user for information until they type QUIT for example. The Python while loop is ideal for this.

Count Controlled Loops¬∂

Sometimes you have a problem that involves doing a certain number of actions. You have a counter that counts up or down until you reach the end of the problem. We will use the while loop (or even the for loop) and show you how to do this type of problem.


 

 

 

 

 

 

Event Loops¬∂

The while loop is often used when you have an event that might stop it looping. For example, reading input frequently until a special value is read. That is an event that might stop the while loop.
