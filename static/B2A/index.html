From the Bits to Applications¶

1.0 Introduction¶

This book was originally based on material that was developed by me, Bob Kummerfeld, for the University of Sydney School of Computer Science INFO1112 course in 2020. That was the year the pandemic started and we had to rapidly pivot from conventional face-to-face lectures and labs to fully online delivery.

I’ve taken that material, and a lot more, and developed a series of videos for a new youtube channel (youtube.com/@cs-tutorials). I have also started writing this e-book covering the material that includes self-test questions and embedded videos.

To build the e-book I’m using the Sphinx document processor and a preprocessor program that I wrote to add a few extra features.

Eventually I plan to enable a user modelling system with a backend personal user model for each user. Then I will add a simple dashboard display that allows users to see where they are up to in the material, what should be done next, what the main concepts are etc.

The material is arranged in a series of web pages, each with text, graphics, embedded self test exercises and video. The e-book is currently in draft form.
