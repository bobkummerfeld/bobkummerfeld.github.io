Ext1 Characters¶

Almost every time you use a computer you work with characters. You type them in and manipulate them in documents. You work with a range of different characters. even emojis! In this video I describe exactly how characters are represented in the computer.

Ext4 Sets¶

Python has a useful data structure called the set. It has operations that allow you to union, intersect and other mathematical set operations. It is easy to create and manage in Python.


 

 

 

 

 

 

Ext8 Recursive Functions¶

A function that calls itself is a “recursive” function. A recursive solution for a problem can be very elegant but recursive solutions are not strictly necessary.


 

 

 

 

 

 

Ext9 With Statement¶

Sometimes you might need to recover when an error occurs in your code. Python has a way of managing a block of code that depends on some activity such as reading data from a file. This approach lets you fix errors If they occur. The with-statement will do this for you.


 

 

 

 

 

 

Ext17 Numerical Python: Arrays and NUMPY¶

If you want to do some very fast calculation in a Python program, you may have to use a different way of storing data. Python lists are not as efficient or high speed as an array. The NUMPY package implements arrays using the C language and lets you store your data in very fast arrays.
