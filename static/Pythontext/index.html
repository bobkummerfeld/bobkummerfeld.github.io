1.0 Introduction¶

This is an introduction to the Python programming language. It covers the Python language essentials and the more advanced features.

The book is divided into a set of topics and each topic is described in a similar format. Within a topic there is first a description of the aspect of Python. Then an example of the feature. Then a series of exercises to do with that …..
